# Buttons and State — Learn egui in Neovim (Episode 3)

A counter app built with egui/eframe demonstrating interactive buttons and mutable state.

## What You'll Build
- `ui.button()` for creating clickable widgets
- `.clicked()` for detecting button presses
- `ui.horizontal()` for side-by-side layout
- Mutable state in the App struct (`counter: i32`)
- Two-file structure with `mod` and `use`

## Prerequisites
- [Rust](https://rustup.rs/) installed
- Completed [Episode 2: Labels and Headings](https://github.com/GoCelesteAI/egui_labels_and_headings)

## Run
```bash
cargo run
```
